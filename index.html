PS4

The Play Station 4 is the latest console made by Sony. With it's ability to be upgraded,
It is Microsofts Xbox One's biggest competitor.
More information

Pros

-Removable hard drive for upgrading.
-Games have great graphics for the price.
-Improved social features like the What's New tab, which is like Facebook for PSN.
-The controller is better, with improved triggers, a speaker, analog sticks and the touchpad is awesome (some will disagree)

Cons

-The OS can be glitchy. Sometimes the PS Store or friends list takes awhile to load. Things get jittery sometimes if you use the web browser to watch videos for a long time.
-Sometimes the machine gets a disk stuck in it and you have to manually eject it.
-Sometimes it will give you issues if your electricity goes out while the console is in standbye mode.
-The controller has a short battery life.

ALIENWARE ALPHA CONSOL

The Alienware Alpha Consol is a new consol/desktop with its new dual mode.
When you launch the consol up you can choose between
desktop mode or consol mode.
The Consol mode has a special interface specifically for the consol,
and the desktop mode is a standard traditional desktop computer.

Pro's

-There are multiple versions.
-The only difference from the second most expensive and the most expensive is that it has an i7 processor.
-Better graphics cards than the xbox1 and ps4.
-Has a bigger variety of games at more reasonable costs.
-All steam games,
-All other games since the alpha console has a desktop computer mode too!
-Comes with an xbox 360 controller.
-You can upgrade your hard drive to an ssd (solid state drive)

Con's

-150$ more than a ps4 or xbox1
-Have to buy other accessories depending on what mode you want to use
1. Console mode
a. Wireless headphones for talking in game (optional)
b. TV
c. ssd to start it up faster and run faster if turning it off all the time like a console. (optional depending on your patience level).
d. Only can use 4 controllers at a time (because there are 4 usb ports to put in the cord or a wireless receiver).
2. Desktop mode.
a. Mouse
b. Keyboard (if you don't want to use the ones supplied with the computer).
c. Monitor
d. Headphones (with a headphone jack converter to usb (because there is no headphone jack connector point)).
e. Can't change or upgrade graphics card due to the size of the computer/console itself.
